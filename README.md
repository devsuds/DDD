# About

<p> This is a typical whiteboard application for design and development purposes. The name DDD is acronym for Draw, Design, Develop.
</p>

## Purpose

<p> A simple, yet powerful tool for creating and sharing designs, diagrams, and other visual content.
</p>

## Created using

- React
- TypeScript
- Vite
- tldraw

## Running on

- BUN

## Installation

### Install BUN runtime (if not already installed)

- MacOS/Linux

  ```bash
  curl -fsSL https://bun.sh/install | bash
  ```

- Windows

  ```bash
  powershell -c "irm bun.sh/install.ps1 | iex"
  ```

### Insall dependencies

```bash
bun install
```

## Running the application

```bash
bun run dev
```

## Building the application

```bash
bun run build
```

## Running tests

```bash
bun run test
```
